# Web POS Frontend

Frontend do sistema POS web construído com React, TypeScript e Vite.

## Executando localmente

Pré-requisitos:

- Node.js 20.19 ou superior
- npm

Passos:

```bash
cd frontend
npm install
npm run dev
```

Aplicação local:

- normalmente em `http://localhost:5173`
- se a porta padrão estiver ocupada, o Vite usará a próxima porta disponível e mostrará a URL no terminal

## Scripts úteis

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
